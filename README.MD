# LiteNote

一个简洁的 Android 取件码等相关应用,支持桌面小部件功能。

## 功能特点

- 支持基本的笔记创建、编辑和删除
- 提供产品管理功能
- 支持桌面小部件显示笔记内容
- 支持深色模式
- 支持 Android 12+ 系统

## 技术栈

- Kotlin
- Jetpack Compose
- Room Database
- Glance (用于小部件)
- Material Design 3

## 系统要求

- Android SDK 29+ (Android 10.0 及以上)
- 最高支持 Android SDK 35

## 构建与运行

1. Clone 项目到本地
2. 使用 Android Studio Hedgehog 或更高版本打开项目
3. 同步 Gradle 依赖
4. 运行项目到模拟器或实机

## 项目结构

主要代码结构:

kotlin:app/src/main/java/com/example/litenote
desktopwidget/ # 桌面小部件相关代码
widget/ # 应用内组件
utils/ # 工具类
